# MicroVQA-PlusPlus
Official repository of paper "MicroVQA++: High-Quality Microscopy Reasoning Dataset with Weakly Supervised Graphs for Multimodal Large Language Model"

Paper available at https://arxiv.org/abs/2511.11407.

Dataset available at huggingface https://huggingface.co/datasets/ieellee/MicroVQA_PlusPlus.
